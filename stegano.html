Stegano

Ce programme a été  réalisé en C  dans le cadre de mes études. Le but de ce programme est de pouvoir cacher un fichier/une chaîne de caractère dans des images.

Pour cela, on vient insérer les bits de notre fichier/chaîne de caractère dans le dernier bit de chaque pixel, aussi appelés  "bits faibles" , de notre image, permettant de rendre cette transformation imperceptibe.

Plusieurs techniques de  dissimulation et de chiffrement  sont proposées pour rendre le décodage plus complexe : table de permutation liée à une clé, dissimulation selon un sens de parcours lié à une clé ou à une matrice de Hamming (insertion par syndrome).
